feat(arv): appraiser-style ARV report — range + narrative + comps w/ layout & status

Adopt the ChatGPT-style concept: Gemini now produces a full ARV report, not a
single number.

estimated_arv_low, estimated_arv_high (the RANGE), arv_narrative (2-3 sentence
appraiser rationale: the $/sqft band for renovated comparable-footprint homes

status: Sold|Active|Estimate, value }). The prompt instructs Gemini to act as a

(or range midpoint) still drives MAO + the market-value fallback.

midpoint noted, the full narrative, and a Comparable Properties table with
Address / Layout / Sq.Ft. / Status (color-coded pill) / Value / $/sqft + a
median $/sqft footer. Prefers Gemini's appraiser comps, falls back to raw
provider comps. Offer PDF mirrors the same report.
