Database schema

Every tenant table carries organization_id. RLS uses current_org_id() + has_perm() helpers to gate every row. Service-role bypasses RLS — used only by trusted server routes.

Core tenant tables

  • organizations — tenant root (name · slug · plan_tier · qa_persona · qa_killers)
  • profiles — 1:1 auth.users; carries org_id + role + full_name/username/phone/website + can_download_calls
  • roles + role_permissions — DB mirror of RBAC matrix
  • teams + team_members — group sub-users, leader_id
  • property_data_cache — server-only Zillow result cache (no RLS policies = locked to service role)

Lead lifecycle

  • leads — extracted_address · status · stage · arv · arv_confidence · submission_date (EST trigger) · metadata jsonb
  • lead_status_history — every status change with actor
  • lead_events — timeline log (created · status_changed · stage_changed · call_uploaded · ...)
  • calls — secure recording metadata (private bucket path)
  • campaigns · cold_callers · submission_forms — legacy + org-scoped

Key DB functions & triggers

  • current_org_id() · current_app_role() · has_perm(text) — RLS helpers
  • leads_fill_org (BEFORE INSERT) — auto-derives org_id from owner so legacy inserts work
  • leads_submission_date_guard (BEFORE INSERT/UPDATE) — stamps EST date, blocks edits unless lead.date.override
  • leads_audit (AFTER INSERT/UPDATE) — writes lead_status_history + lead_events
  • touch_updated_at (BEFORE UPDATE) — updated_at clock

Service layer (vendor-agnostic)

services/propertyDataProvider.ts

Abstraction over property-data vendors. Default MockPropertyDataProvider; production uses RapidApiZillowProvider (private-zillow.p.rapidapi.com). Swap via PROPERTY_PROVIDER env.

services/arv.ts

Pure heuristic ARV: comp PPSF → trimmed mean (drops outliers) → × condition × ZIP multipliers. Returns { estimatedArv, confidence, pricePerSqft, compsUsed, breakdown }.

Storage upload policy

Required for sub-users to upload to the private call-recordings bucket.

create policy "call-recordings upload by org"
  on storage.objects for insert to authenticated
  with check (
    bucket_id = 'call-recordings'
    and (storage.foldername(name))[1]
        = public.current_org_id()::text
  );
